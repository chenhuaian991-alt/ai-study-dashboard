# AI Study Dashboard 手动验收清单

## 前置条件

- `npm run dev` 启动项目
- 浏览器打开 http://localhost:5173
- 打开 DevTools → Application → Local Storage 确认数据

---

## 一、任务看板

### 新增任务
- [ ] 填写标题和课程，点击"添加任务" → 任务出现在"待做"列
- [ ] 不填标题直接提交 → 显示红色错误提示"请输入任务标题"
- [ ] 不填课程直接提交 → 显示红色错误提示"请输入课程名称"
- [ ] 填写截止日期、优先级、预计用时、资料链接 → 卡片正确显示
- [ ] 资料链接为空时 → 卡片不显示"资料链接"
- [ ] 资料链接不为空时 → 点击可跳转

### 编辑任务
- [ ] 点击任务卡片的"编辑" → 表单自动填入原数据，顶部出现黄色横幅
- [ ] 修改标题后点"保存修改" → 卡片更新，表单恢复新增模式
- [ ] 点"取消" → 表单清空，退出编辑模式
- [ ] 编辑状态下点其他任务的"编辑" → 表单切换到新任务

### 删除任务
- [ ] 点击"删除" → 弹出确认框，文案包含任务标题
- [ ] 点"取消" → 任务仍在
- [ ] 点"确定" → 任务消失

### 状态切换
- [ ] 待做任务点"开始" → 移到"进行中"列
- [ ] 进行中任务点"完成" → 移到"已完成"列
- [ ] 进行中任务点"退回待做" → 移回"待做"列
- [ ] 已完成任务点"重新打开" → 移回"进行中"列

### 看板布局
- [ ] 桌面端三列横向排列
- [ ] 窗口宽度 < 640px 时纵向排列
- [ ] 无任务时显示"还没有任务"提示
- [ ] 某列无任务时显示"暂无任务"

---

## 二、AI 编程日志

### 新增日志
- [ ] 填写所有必填字段，点"添加日志" → 日志卡片出现在列表
- [ ] 不填标题提交 → 显示错误提示
- [ ] 不填提示词提交 → 显示错误提示
- [ ] 不填结果提交 → 显示错误提示
- [ ] 选择不同工具（Claude/Codex/其他）→ 卡片显示对应标签
- [ ] 选择不同状态（成功/部分成功/失败）→ 卡片显示对应颜色标签
- [ ] 填写经验教训 → 卡片显示 Lesson 字段
- [ ] 不填经验教训 → 卡片不显示 Lesson 字段

### 删除日志
- [ ] 点击"删除" → 弹出确认框，文案包含日志标题
- [ ] 点"取消" → 日志仍在
- [ ] 点"确定" → 日志消失

### 日志列表
- [ ] 无日志时显示"还没有日志"提示
- [ ] 多条日志按添加顺序排列

---

## 三、每日复盘

### 新增复盘
- [ ] 日期默认为今天
- [ ] 填写完成内容和明日计划，点"添加复盘" → 卡片出现在列表
- [ ] 不填完成内容提交 → 显示错误提示
- [ ] 不填明日计划提交 → 显示错误提示
- [ ] 填写延期事项 → 卡片显示"延期"字段
- [ ] 不填延期事项 → 卡片不显示"延期"字段
- [ ] 填写备注 → 卡片显示"备注"字段
- [ ] 不填备注 → 卡片不显示"备注"字段
- [ ] 日期显示为中文格式（如"2026年6月7日 周六"）

### 删除复盘
- [ ] 点击"删除" → 弹出确认框，文案包含复盘日期
- [ ] 点"取消" → 复盘仍在
- [ ] 点"确定" → 复盘消失

### 复盘列表
- [ ] 无复盘时显示"还没有复盘记录"提示

---

## 四、JSON 导入导出

### 导出
- [ ] 点击"导出备份" → 浏览器下载 `.json` 文件
- [ ] 点击"复制备份 JSON" → 提示"备份 JSON 已复制"，粘贴验证内容正确
- [ ] 点击"显示备份 JSON" → 下方出现 textarea，内容为完整 JSON
- [ ] textarea 中点击 → 自动全选

### 导入
- [ ] 选择 `.json` 文件，点"导入备份" → 弹出确认框
- [ ] 点"取消" → 不导入
- [ ] 选择有效备份文件，点"确定" → 提示导入成功，页面自动刷新，数据恢复
- [ ] 选择格式错误的文件 → 显示错误信息，当前数据不受影响
- [ ] 选择缺少字段的文件 → 显示具体错误信息

## 五、localStorage 持久化

- [ ] 添加任务后刷新页面 → 任务仍在
- [ ] 编辑任务后刷新页面 → 修改仍在
- [ ] 切换任务状态后刷新页面 → 状态仍在
- [ ] 添加日志后刷新页面 → 日志仍在
- [ ] 添加复盘后刷新页面 → 复盘仍在
- [ ] 删除任意数据后刷新页面 → 数据仍不存在
