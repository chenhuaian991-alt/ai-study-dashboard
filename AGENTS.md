# AGENTS.md

## 项目背景

这是一个面向大一学生的本地 AI 学习任务管家，用于管理课程任务、AI 编程日志和每日复盘。

## 通用启动指令

每次开始新任务时，先阅读：

1. AGENTS.md
2. PROJECT_PLAN.md
3. README.md

如果存在 CONTEXT.md，也要先阅读它了解之前的工作上下文，然后继续工作，不要重复已完成的步骤。

CONTEXT.md 固定放在项目根目录。

## 技术栈

- Vite
- React
- TypeScript
- 普通 CSS
- localStorage

## 第一版范围

- 任务看板
- 任务新增、编辑、删除
- 任务状态切换
- AI 编程日志
- 每日复盘
- localStorage 本地保存
- JSON 导入导出

## 暂不实现

- 登录
- 后端
- 数据库
- 云同步
- AI API 接入
- 移动 App
- 多用户权限
- 复杂统计报表
- 自动提醒

## 通用开发原则

- 优先完成可用 MVP
- 每次只完成一个明确功能
- 不主动扩大需求
- 不主动引入复杂依赖
- 不做无关重构
- 遵循现有项目结构和命名风格
- 修改前先说明计划
- 修改后说明改了哪些文件、实现了什么、如何验证

## 代码质量要求

- TypeScript 类型要清晰
- 组件职责要单一
- 数据结构要易于理解
- 不把所有逻辑堆进一个大文件
- 用户数据保存和恢复逻辑要谨慎处理
- 对导入 JSON 等容易出错的功能要有错误处理

## 存档指令

当用户说“存档”或“保存上下文”时：

1. 把当前对话的关键信息压缩写入工作区的 CONTEXT.md
2. 必须用覆盖模式写入，不要追加
3. 控制在 20 行以内
4. 使用中文
