# AI Study Dashboard

AI Study Dashboard 是一个面向大一学生的本地学习任务管理工具，用来管理课程任务、AI 编程日志和每日复盘。

项目的第一版目标是做出一个可以长期使用的本地 Web 应用，而不是追求复杂架构或炫技功能。

## 第一版功能（v0.1）

- 任务看板：管理待做、进行中、已完成的学习任务
- 任务信息：记录课程、截止日期、优先级、预计用时和资料链接
- AI 编程日志：记录使用 Claude Code 和 Codex 进行开发的过程、结果和问题
- 每日复盘：记录当天完成内容、延期任务、明日计划和备注
- 本地保存：使用 localStorage 保存数据
- 数据导入导出：支持 JSON 备份和恢复

## v0.2 功能

- 首页概览 Dashboard：今日到期、已逾期、进行中任务一览，最近 AI 日志和每日复盘摘要
- 任务搜索与筛选：按关键词、状态、优先级、课程筛选任务，支持清空筛选

## v0.3 功能

- AI 日志筛选：按关键词、工具、状态筛选 AI 编程日志
- 复盘材料生成：根据筛选结果自动生成 Markdown 复盘材料，一键复制给 AI 总结
- AI 复盘摘要：保存 AI 返回的总结结果，支持查看和删除

## 暂不实现

- 登录系统
- 后端服务
- 数据库
- 云同步
- AI API 接入
- 移动 App
- 多用户权限
- 复杂统计报表
- 自动提醒

## 技术方向

- Vite
- React
- TypeScript
- 普通 CSS
- localStorage

## 开发原则

- 优先完成可用的 MVP
- 每次只实现一个明确功能
- 不主动扩大需求范围
- 不引入不必要的复杂依赖
- 页面以清晰、稳定、适合长期使用为主
- 每次修改后说明改了什么，以及如何验证

## 后续扩展方向

- 更完整的任务统计
- 复习提醒
- 周报和月报
- 更好的搜索和筛选
- 可选的云同步
- 可选的 AI 总结功能

## v0.3.1 功能

- AI 复盘摘要保留多段换行显示。
- 日志筛选主要用于生成复盘材料，不影响正常日志列表查看。
- AI 复盘摘要可绑定来源日志，并在摘要卡片中展示来源。
- 首页概览支持快速跳转到任务、AI 日志和每日复盘区域。
- AI 日志页提供页内快捷定位，便于在新增日志、复盘材料、日志列表和已保存复盘之间移动。
- JSON 备份继续兼容旧数据，并补齐 AI 复盘摘要的来源日志字段。
