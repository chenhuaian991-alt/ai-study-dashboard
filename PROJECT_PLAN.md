## 项目目标

做一个本地运行的 AI 学习任务管家，帮助大一学生管理课程任务、AI 编程日志和每日复盘。

## MVP 范围

第一版只实现本地可用功能：

- 任务看板
- 任务新增、编辑、删除
- 任务状态切换：待做、进行中、已完成
- AI 编程日志
- 每日复盘
- localStorage 本地保存
- JSON 导入导出

## 非目标

第一版不做：

- 登录
- 后端
- 数据库
- 云同步
- AI API 接入
- 移动 App
- 多用户系统
- 复杂权限
- 自动提醒

## 技术栈

- Vite
- React
- TypeScript
- 普通 CSS
- localStorage

## 开发顺序

1. 创建项目启动文档

3. 搭建基础页面布局

5. 实现任务看板
6. 实现 localStorage 本地保存
7. 实现 AI 编程日志
8. 实现每日复盘
9. 实现 JSON 导入导出
10. 打磨 UI 和空状态

12. 写开发复盘文档

## 第一版验收标准

- 可以新增、编辑、删除任务

- 刷新页面后任务不丢失
- 可以记录 AI 编程日志
- 可以记录每日复盘

- 页面没有明显布局错乱
