# PROJECT_PLAN.md

## 当前阶段

阶段 0：项目启动与规则建立。

本阶段只创建项目说明和 AI 协作规则，不初始化前端项目，不安装依赖，不创建业务代码。

阶段 1 从初始化 Vite + React + TypeScript 项目开始，目标是得到一个能运行的空壳项目、基础目录结构和核心类型定义。

## 项目目标

做一个本地运行的 AI 学习任务管家，帮助大一学生管理课程任务、AI 编程日志和每日复盘。

## MVP 范围

第一版只实现本地可用功能：

- 任务看板
- 任务新增、编辑、删除
- 任务状态切换：待做、进行中、已完成
- AI 编程日志
- 每日复盘
- localStorage 本地保存
- JSON 导入导出

## 非目标

第一版不做：

- 登录
- 后端
- 数据库
- 云同步
- AI API 接入
- 移动 App
- 多用户系统
- 复杂权限
- 自动提醒

## 技术栈

- Vite
- React
- TypeScript
- 普通 CSS
- localStorage

## 开发顺序

1. 创建项目启动文档
2. 初始化 Vite + React + TypeScript 项目
3. 搭建基础页面布局
4. 定义任务、AI 日志、每日复盘的数据模型
5. 实现任务看板
6. 实现 localStorage 本地保存
7. 实现 AI 编程日志
8. 实现每日复盘
9. 实现 JSON 导入导出
10. 打磨 UI 和空状态
11. 编写基础测试或手动验收清单
12. 写开发复盘文档

## 第一版验收标准

- 可以新增、编辑、删除任务
- 可以切换任务状态
- 刷新页面后任务不丢失
- 可以记录 AI 编程日志
- 可以记录每日复盘
- 可以导出 JSON 数据
- 可以从 JSON 数据恢复
- 页面没有明显布局错乱
- 功能边界符合 README 和 AGENTS.md
